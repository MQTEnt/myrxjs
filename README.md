### Learning RxJS:
Using JavaScript ES6 by Babel

### .pipe(operator1, operator2,...)
* Filter: `filter`, `take`
* Flattening operators: `map`, `mergeMap`, `concatMap`
* Log: `tap`, `do`
* Delay: `delay`
* Combination: `merge`, `mergeAll`, `forkJoin`
* Error Handing: `catchError`
